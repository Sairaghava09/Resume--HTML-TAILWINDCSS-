# Resume – HTML + Tailwind CSS

A clean, responsive personal resume/portfolio webpage built using HTML and Tailwind CSS. This project provides a modern, customizable CV template that you can host online or modify for your own professional use.

---

## Overview

This project is a visually appealing resume website created with:

- HTML — for the structure and layout
- Tailwind CSS — for utility-first styling and responsive design

It allows you to showcase your skills, education, experience, and contact information in a web-ready format.

---

## Features

- Modern, professional resume layout
- Fully responsive design for desktop and mobile
- Easy to customize sections like skills, projects, and experience
- Can be hosted on GitHub Pages or any static hosting platform
